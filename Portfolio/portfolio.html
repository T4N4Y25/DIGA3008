    Cyclone Arcade Game

    Project Report

    Read Report Download Report

    The popular cyclone arcade game is made using the Arduino Uno Microcontroller and is programmed entirely using AVR assembly! A large emphasis on this project was using timers and interrupts through a strolling LED sequence where the user can press a button to try match the strolling light with a randomly chosen LED light. This was an individual project which took around two months in 2024 to complete for the course, Microprocessors.

    Clap-Detecting Circuit

    Project Report

    View Report Download Report

    The Clap-Detecting circuit is a circuit with no software used; only hardware. A microphone measures claps as inputs and subsequentally counts them on a display. The circuit acts a subsystem for a larger circuit known as the Crypto-Box. When a correct number of claps is detected, the circuit allows progression to the next sub-system involving a Piezo sensor (monitors taps on the sensor as inputs) which was done by a fellow group member for the project. My role in the Crypto-Box project was to create the circuit which counts claps as inputs using an electret microphone and indicate when the correct number of claps is detected (four claps) using an LED. The project was successfully completed and met all the requirements.
